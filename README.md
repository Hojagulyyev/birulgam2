# BirUlgam: cloud-based ERP/CRM software.

## Prerequisites
- python3.10
- asyncpg
- fastapi
- redis
- poetry
- strawberry

## Installation

Note: Use the dev branch for installation and contributing.

```
git clone https://github.com/Hojagulyyev/birulgam2.git
cd birulgam2
cp .env.example .env
vim .env
```
and setup the .env
